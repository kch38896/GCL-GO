# GCL-GO
This repository contains the code for the paper: 

GCL-GO: A novel sequence-based hierarchy-aware method for protein function prediction

Our paper is accepted in BIBM 2022.
## Dependencies
+ python
+ PyTorch
+ SciPy
+ scikit-learn
+ NumPy
+ pandas
